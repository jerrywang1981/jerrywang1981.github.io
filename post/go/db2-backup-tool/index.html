Db2 Backup Tool

2020年4月25日
| it golang db2

为什么要做这个小工具

有时候，我们可能要把数据库一些表的数据导出，然后load到另外一个数据库，虽然有很多的工具可以选择， 但是可能用的不太顺手。所以就想，是不是能够自动生成export.sql文件。基本上来说，就是通过table，查询到 表结构，把所有的列选择出来，export到del文件，在生成的另外一个import.sql, 然后load client。 当然，生成的内容，可能不是100%的符合要求，还是要求用户去检查一下，以确保正确。

代码

  • go_backup_db2

用法

目前还是命令行的形式，需要提供很多的参数，因为需要db2数据库的所有信息。
