Angular Javascript Heap Out of Memory

2020年5月14日
| it node angular

如果启动angular的时候有报 Javascript heap out of memory错误，可以试着修改

node_modules/.bin/ng

加上 --max_old_space_size

#!/usr/bin/env node --max_old_space_size=4096
